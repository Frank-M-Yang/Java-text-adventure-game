### Installation
```bash
javac src/game/main/Main.java
java game.main.Main
```

### Commands

| Command | Example |
|---------|---------|
| `go <direction>` | `go east`, `go e` |
| `look [target]` | `look`, `look chip` |
| `status` | Check progress |
| `help` | Show commands |
| `quit` | Exit |

### Walkthrough
